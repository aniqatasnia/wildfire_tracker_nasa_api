# Wildfire Tracker

> This is a React app to track wildfires and plot them on a Google map.

## Usage

```
npm install
npm install @vis.gl/react-google-maps @iconify/react @iconify/icons-mdi
npm start

## API Keys

You will need to add your Google Maps API key in the Map.js component. If you deploy, please move the keys to a safe place (see .env.example).
