# Wildfire Tracker

> This is a React app to track wildfires and plot them on a Google map.
> To view a demo of this project, go to https://aniqatasnia-demo-wildfire-tracker.netlify.app/  
> *Note that due to the demo Google Maps API, a popup will display from the browser. Dismiss the popup by clicking "OK".*

## Usage

```
npm install
npm install @vis.gl/react-google-maps @iconify/react @iconify/icons-mdi
npm start

## API Keys

You will need to add your Google Maps API key in the Map.js component. Additionally, see .env.example.
